### 网站论坛系统

#### 前端：
前端页面模仿了思否的前端页面风格
（https://segmentfault.com/）

使用技术栈如下：
- bootstrap4
- jquery
- CKEditor4-full-version
- thymeleaf 3.x

#### 后端：
使用技术栈如下：
- springboot 2.2
- mybatis
- 数据库：mysql
- 文件上传服务：fastdfs

（如果需要使用文件上传功能，需要自行配置fastdfs服务，最终修改application.properties
中的对应内容即可）

（目前我已经在我自己的服务器有用docker部署过了，这是项目入口的地址：http://121.36.19.240:8081）
